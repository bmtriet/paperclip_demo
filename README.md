# Paperclip Prototype v1

This repository contains a static website prototype for the Paperclip product direction review.

## Page Roles
- `index.html` - visual direction, prototype intent, and page structure overview
- `features.html` - capability pillars, interaction model, and handoff notes
- `pricing.html` - commercial structure and rollout tiers
- `contact.html` - prototype feedback and demo request form

## Shared Assets
- `assets/css/main.css` - shared design system, layout, and responsive rules
- `assets/js/main.js` - active nav state, year stamp, and reveal-on-scroll behavior

## Run Locally
1. From repo root, run `npm run start`.
2. Open `http://localhost:4173`.

## Notes
- The prototype is frontend-only and intentionally avoids backend dependencies.
- The layout is built to support product review, IA validation, and engineering handoff.

## Content Follow-up
- Marketing/content should own the final pass on hero headlines, feature descriptions, pricing labels, and contact prompts.
- The current copy is directional and supports the structure, but it still needs an editorial pass for tone consistency and sharper product messaging.
- Staffing for that content function is being resolved in the separate CMO task; the highest-risk area remains the home-page hero and the pricing page, where wording drives perception most strongly.
